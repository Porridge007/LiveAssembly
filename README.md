# LiveAssembly产品逻辑
## 概述
本产品分为推流两个模块。  
推流模块提供推流的地址，用户向其推送rtmp流、rtsp流，程序通过goroutine启用go协程并发调用livego对直播地址进行分发。  
拉流模块由客户提供的拉流地址，程序使用ffmpeg将rtsp流转为rtmp流代为推送，并由live共进行分发。
## 具体细节
推流模块：
1.程序启动
goroutine启动livego当作服务  
2.新建推流  
【用户】   
1.点击新增  
2.向推流地址推流  
3.从观看地址进行收看  
【程序】 
1. 返回推流地址和观看地址。创建channel，发送请求，新建房间，房间后缀自增。livego将接收并分发。
